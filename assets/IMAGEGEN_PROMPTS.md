# Imagegen Prompts

These repository visuals were generated with the built-in Codex `image_gen`
tool on 2026-06-05.

The images intentionally avoid embedded text. README copy, badges, headings,
and release-status wording stay in Markdown so the repository remains readable,
localizable, and easy to correct.

