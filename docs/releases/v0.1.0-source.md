## Included

- Sanitized Android and Windows client source.
- Community client mode for local keys and subscription URLs.
- Android QR camera import through `mobile_scanner`.
- Windows QR camera import through `camera`, `camera_windows`, and `zxing2`.
- Manual subscription URL import plus foreground/manual refresh. Later source
  snapshots may add in-app scheduled refresh; this does not imply an OS
  background service.
- Gated Free VPN catalog candidate metadata for `AvenCores/goida-vpn-configs`.
- GitHub Actions CI and clean-clone source-boundary proof.

## Not Included

- No APK or EXE binaries.
- No store release.
- No trusted Windows signing claim.
- No official POKROV backend, billing, admin, deployment, signing, or private
  release evidence.
- No enabled-by-default third-party public config feeds.

## Known Limitations

- Source-only release: users must build from source.
- Runtime artifacts, private deployment details, signing identities, and
  official release evidence are not committed.
- Free VPN catalog metadata is gated and disabled by default.
- Subscription refresh is foreground/manual in this tag; later `main` commits
  add an in-app foreground scheduler without claiming OS background refresh.
- Dependency license audit is direct-dependency focused; exact transitive
  inventory remains a follow-up before binary release claims.
