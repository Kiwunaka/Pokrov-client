# Brand Policy

The POKROV client code is open source under the repository license. The POKROV
brand is governed separately.

## Official POKROV Assets

The POKROV name, logos, domains, Telegram channels, support bots, signing
identities, and official release channels identify builds and services operated
by POKROV.

## Forks And Rebuilds

You may fork and modify the code under the repository license, but you must not:

- imply that a fork is an official POKROV build
- use POKROV signing identities
- use POKROV domains or support bots as if they belong to your fork
- copy official release-readiness claims without your own evidence
- mislead users about who operates the backend service

## Recommended Fork Wording

Use wording such as:

> This is an independent fork of the open-source POKROV client. It is not an
> official POKROV build.

## Official Builds

Official binaries are published only through POKROV-owned release channels.
Users should verify release links, version labels, and checksums before
installing.
